# CRUD de Clientes en Java

Este es un sistema de gestión empresarial desarrollado como parte de mi portafolio profesional.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Java 21 (Ecosistema robusto)
* **Framework Backend:** Spring Boot (Inyección de dependencias y configuración automática)
* **Persistencia:** Hibernate / JPA conectado a una base de datos **SQLite**
* **Interfaz Gráfica:** JavaFX con diseño declarativo en **FXML**
* **Gestor de Construcción:** Maven
* **IDE:** Eclipse
